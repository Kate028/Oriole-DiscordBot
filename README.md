
# Oriole DiscordBot
 
 **An open source multipurpose discord bot.**


# Installation
**Important Notes:**
 - Python 3.9 or higher is recommended
 - Bot is running on discordpy 2.0a 

1. Install dependencies 
```
$ pip install -r requirements.txt
```
2. Create .env file
```
TOKEN= #Your bot's token
```
3. Run the bot
```
$ pyhton launcher.py
```
