<!-- Installation -->
 ## Getting Started

  **Instructions on setting up Oriole DiscordBot locally.**

  ### Prerequisites

   - Python 3.9 or higher is recommended
   - Bot is running on discordpy 2.0a 

  ### Installation

  1. Install dependencies 
   ```
   $ pip install -r requirements.txt
   ```

  2. Create .env file
   ```
   TOKEN= #Your bot's token
   ```

  3. Run the bot
   ```
   $ python launcher.py
   ```
